# S3 - La méthode SARDE de résolution de conflit

**Type :** E-learning
**Durée :** ~35 min
**Statut :** ✅ Complété

## Points clés à retenir

1. **Le conflit est normal et sain** : L'absence de conflit dans une équipe est souvent le signe que personne n'ose s'exprimer, pas que tout va bien. L'enjeu n'est pas d'éviter les conflits mais de les résoudre constructivement.

2. **La méthode SARDE** : Cadre structuré pour aborder un conflit interpersonnel :

| Étape | Signification | Action |
|-------|---------------|--------|
| **S** | Situation | Décrire les faits objectivement, sans interprétation |
| **A** | Analyse | Identifier les besoins non satisfaits de chaque partie |
| **R** | Ressenti | Exprimer les émotions associées (sans accusation) |
| **D** | Demande | Formuler ce qu'on souhaite comme changement |
| **E** | Engagement | Définir ensemble les actions concrètes |

3. **Avant d'utiliser SARDE** : Choisir le bon moment et le bon cadre. Un conflit ne se règle pas en public, en urgence, ou quand les émotions sont trop vives. Créer les conditions d'une conversation constructive.

4. **Les pièges classiques à éviter** :
   - Confondre faits et interprétations ("Tu m'as ignoré" vs "Tu n'as pas répondu à mon message")
   - Régler les comptes du passé au lieu de se concentrer sur le problème actuel
   - Chercher à avoir raison plutôt qu'à trouver une solution

5. **Le rôle du chef de projet dans les conflits d'équipe** : Il est rarement partie prenante directe. Son rôle est de créer les conditions pour que les personnes concernées se parlent, pas de décider à leur place.
