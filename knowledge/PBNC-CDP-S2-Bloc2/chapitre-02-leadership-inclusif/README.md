# Chapitre 2 - Le leadership inclusif et à distance (15h)

**Statut :** ✅ Terminé (14/14)

## Résumé du chapitre

Piloter un projet no-code nécessite souvent de coordonner des équipes hybrides, à distance, avec des profils très variés. Ce chapitre outille le chef de projet pour évaluer les compétences disponibles, organiser la gouvernance, animer les rituels d'équipe et gérer les conflits avec assertivité.

## Concepts clés

- **Matrice des compétences** : outil pour cartographier les compétences disponibles vs nécessaires
- **Biais cognitifs** dans l'évaluation des compétences (tendance à surestimer/sous-estimer)
- **Document de cadrage** : la "partition commune" qui aligne toute l'équipe sur les règles du jeu
- **Rituels agiles** : daily, sprint review, rétrospective — leur mise en place à distance
- **Styles de communication** : assertif, passif, agressif, passif-agressif
- **Méthode SARDE** : outil structuré de résolution de conflit
