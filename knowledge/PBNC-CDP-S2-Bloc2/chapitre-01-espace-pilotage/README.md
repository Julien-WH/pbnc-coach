# Chapitre 1 - L'espace de pilotage du projet (15h)

**Statut :** ✅ Terminé (12/12)

## Résumé du chapitre

Un projet bien piloté, c'est un projet où le chef de projet voit ce qui se passe AVANT que ça devienne un problème. Ce chapitre construit les trois instruments essentiels du cockpit : la roadmap agile (où va-t-on ?), le budget dynamique (combien ça coûte vraiment ?) et le dashboard de pilotage (est-ce qu'on est sur la bonne trajectoire ?).

## Concepts clés

- **Cockpit de pilotage** : tableau de bord centralisant toutes les informations nécessaires à la prise de décision
- **Budget photo vs film** : le budget statique (photo = instantané) vs le budget dynamique (film = évolution continue)
- **OKRs** : Objectives and Key Results — méthode de fixation d'objectifs mesurables
- **Roadmap agile** : planification flexible qui s'adapte aux découvertes en cours de projet
- **KPIs vs métriques** : indicateurs qui influencent les conversions vs données brutes non actionnables
